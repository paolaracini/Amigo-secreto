<h1> 🎁 Amigo Secreto </h1>

Una aplicación web simple y divertida para organizar tu juego de Amigo Secreto ✨

Características:
<ul>✅ Agrega nombres de tus amigos a una lista.

✅ Validación de entradas vacías.
✅ Interfaz amigable y fácil de usar.</ul>
